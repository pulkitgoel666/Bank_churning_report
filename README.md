# Bank_churning_report

Data Analysis Project on Account Closures
Overview

This project aims to analyze account closure trends among customers, identifying key demographics and underlying issues that influence account retention. Through insightful data visualizations and statistical analysis, the project seeks to provide actionable recommendations for improving customer engagement and retention strategies.
Project Structure

    /data: Contains original and cleaned datasets used for analysis.
    /notebooks: Jupyter notebooks containing data exploration, analysis, and visualization scripts.
    /visuals: Directory for generated charts and graphs used in the presentation.
    /reports: Finalized presentation slides and documentation files.
    README.md: Project overview and instructions.

Key Findings

    Demographics: Analysis revealed that 54.5% of account closures were from male account holders.
    Age Groups: A significant closure rate was noted among customers aged 30-60.
    Geographic Trends: Nearly 50% of closures originated from customers in France.
    Credit Score Impact: A notable percentage of account closures occurred among individuals with fair and low credit scores.

Recommendations

    Targeted Engagement: Develop marketing strategies focused on male customers.
    Geographic Accessibility: Expand branch availability in areas with high closure rates.
    Educational Programs: Implement initiatives for credit management education to support customers.
    Feedback Mechanisms: Establish channels to gather feedback from customers.
